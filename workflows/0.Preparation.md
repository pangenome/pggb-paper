# Preparation

## Data compression

```bash
cd /lizardfs/guarracino/pggb-paper/assemblies/

ls *fa | while read FASTA; do
    echo $FASTA
    zstd -12 $FASTA
done

# "zstd -d file.zst" to decompress
```
